• filter_drama Instrucciones

    Instrucciones:
    Se debe crear una aplicación simple en donde se pueda completar uno o más juegos de totito (tic tac toe). Las condiciones son:

    • label_outline Se puede trabajar sobre un framework (p.e. Angular para javascript, Django para python, Play para Java)
    • label_outline Se debe enviar las instrucciones para su ejecución y la carpeta con los archivos comprimidos
    • label_outline No debe ser muy compleja, para que pueda ser terminada en un promedio de dos a tres días

    Las funcionalidades esperadas son:

    • label_outline Se debe comenzar un juego sin necesidad de seleccionar la pieza
    • label_outline El juego es manual, no hay lógica de CPU contra la cual se juega
    • label_outline Se debe poder visualizar qué casillas han sido marcadas por cuál pieza
    • label_outline Se debe poder reiniciar el juego
    • label_outline Al terminar se debe mostrar qué pieza fue la ganadora o si ha sido un empate
